Описание, как нужно получать задания, которые необходимо будет выполнять.

Для выполнения заданий создайте директорию (пакет) со своей фамилией
на английском языке (ОБЯЗАТЕЛЬНО с маленькой буквы)
в пакете со своей группой.

Например:

by.it.group151002.ivanov
by.it.group151002.petrov

В этот пакет копируйте (НО НЕ ПЕРЕМЕЩАЙТЕ) задания из пакета by.it.a_khmelev

Это будут пакеты с именами lesson01 lesson02 и т.д.
Задания нужно выполнить и проверить приведенными тестами (они неполные, приведены как пример).
Тесты СЛЕДУЕТ расширить для проверки крайних случаев и производительности решений.
В прикрепленной библиотеке подключен JUnit 4. Для простоты ограничимся этой библиотекой.
Можно использовать параметрические тесты для объемных случаев.

Все Pull Request-ы для простоты проверки будем собирать в основной ветке main/master

За каждое стирание или изменение ЧУЖОЙ информации
или искажение данных в чужих пакетах (даже 1 байт)
штраф: -1 балл от максимальной итоговой оценки на экзамене.
